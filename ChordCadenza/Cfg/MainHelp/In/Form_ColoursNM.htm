ChordMap Colours

This window can be used to set the colours used by the Chords Panel on the ChordMap.

Configurable Colours

The colours set by this window are:

  • Background: $$frmColoursNM.$cmdBackground
  • Play Cursor: $$frmColoursNM.$cmdPlayCsr
  • Selected Area: $$frmColoursNM.$cmdSelectedArea
  • Keyboard: $$frmColoursNM.$cmdKB
  • Common Chord: $$frmColoursNM.$cmdCommon
