Tonnetz Colours

This window can be used to set the colours used by the Tonnetz Display.

Configurable Colours

The colours set by this window are:

  • Background: $$frmColoursTonnetz.$cmdBackground
  • Diatonic Note: $$frmColoursTonnetz.$cmdDiatonic
  • Chromatic Note: $$frmColoursTonnetz.$cmdChromatic
  • Active Node: $$frmColoursTonnetz.$cmdActive
  • Inactive Node: $$frmColoursTonnetz.$cmdInactive
  • Played Note: $$frmColoursTonnetz.$cmdPlayed
  • New Active Node: $$frmColoursTonnetz.$cmdNewActive
  • Old Active Node: $$frmColoursTonnetz.$cmdOldActive
