PlayMap Colours

This window can be used to set the colours used by the PlayMap.

You should try to set these colours in order to enhance the readability of the PlayMap. This is a personal preference, as everybody has a different perception of colours and contrast.

Some of these colours may overlap each other, so be sure to check what the ChordMap looks like after changing a colour!

Configurable Colours

The colours set by this window are:

  • Main Background: $$frmColoursSC.$cmdMainBackground
  • Black Key: $$frmColoursSC.$cmdBlackKey
  • Chord Note: $$frmColoursSC.$cmdChordNote
  • Root Note: $$frmColoursSC.$cmdRoot
  • Dominant Note: $$frmColoursSC.$cmdDominant
  • Note: $$frmColoursSC.$cmdNote
  • Track Note: $$frmColoursSC.$cmdTrack
  • Bottom Background: $$frmColoursSC.$cmdBottomBackground
  • Played Note: $$frmColoursSC.$cmdPlayed
  • Current Background: $$frmColoursSC.$cmdCurrentBackground
  • Scale: $$frmColoursSC.$cmdScale
