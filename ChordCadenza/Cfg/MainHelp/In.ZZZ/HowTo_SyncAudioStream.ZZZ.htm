How To Synchronise to an Audio Stream

This explains how to sync to an audio stream played by an external source.

This is similar to synchronising to an AudioFile, but with some important differences, highlighted in red.

Load a Project

This should contain at least a ChordFile (*.chp) that is an approximate match to the audio stream that you will be playing. (This means it should have the same rhythm and structure, but doesn't need to be in the same key.)

Setup the Sync SwitchKey

Goto PlayMap Menu > Configure > SwitchKeys/Pedal.

Set the Sync dropdown box to a SwitchKey or Pedal.

Position the ChordFile to a Cue Point

You can use the TrackMap or ChordMap to do this. However, you will need to cue to somewhere you will recognise in the song (e.g. when a solo or vocal starts) that is on a barline.

If you have created a lyric file (*.chl), you can use the TrackMap or ChordMap to show the lyrics, and use them to help find a good cue point.

Otherwise, it's usually easier to use a solo track on the TrackMap to find a cue point.

Set the normal cursor (vertical line) in the TrackMap (click in any track pane) or ChordMap (click in body of Midi pane) to set the cue position. This is where you will start synchronising from. This must be somewhere after where you set the song to start playing from.

Sync to the Audio Stream

Click the Sync Audio Button on any playable window (TrackMap, ChordMap or PlayMap) and start playing the audio stream. Hit the Sync SwitchKey (or Pedal) when the cue point is reached.

The Sync Audio Button on the PlayMap should turn red. Continue to hit the key on every beat until the end.

Hit the Stop Play Button to end the sync.

If you don't get to the end, you can start again from the same or a different cue point and start point.

If you start from a different cue point that overlaps the existing Play Track, it will continue from where you left off. Otherwise, it will set a new start point, and a warning message will be issued.

The sync data will be saved in the AudioSync Record Track. The Play Track will not be updated until you use Copy or Merge from the AudioSync window.

Save the Sync Data

The AudioSync Record Track contains the last sync that you recorded.

If you want to save or play this, you will need to merge or copy it into the AudioSync Play Track.

Open the AudioSync window from PlayMap Menu > File > Show AudioSync Window

Click the Merge Record To Play Button. If you get an error message, it means you are trying to merge data that is not in sequence. You will have to either use Copy Record To Play or re-record the sync data.

You can the use the Save Play to File Button to save the sync data to a .chtx file.

Set Start Time (Optional)

You can set the time of the starting cue point using the Numeric Input controls in the AudioSync Window. This should be copied from the external audio player (assuming it contains a time display).

This can be useful for repositioning to somewhere in the audio stream after the start cue point. Clicking in the Bar Pane on the TrackMap or ChordMap will show the approximate elapsed time, provided the starting cue point time has been entered. You can then use this to reposition the external audio player.

Partial Synchronisation (Optional)

If you want to synchronise from somewhere other than the start (e.g. to correct a mistake, or continue from where you left off), you should position a few seconds before a cue point. This should overlap, or be at the end of an existing synchronisation. You can then do the synchronisation as before to add or replace the sync times.
