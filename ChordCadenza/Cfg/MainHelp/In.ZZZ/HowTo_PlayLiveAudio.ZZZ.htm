How To Play along with Live Audio

You can play along to a live audio stream without synchronising the midi data (chords) beforehand. This could be an internet audio stream, or an audio file played with an external player, or even a live group.

This requires you to use a SwitchKey to move forward to the next beat or chord whilst playing the chords.

Setup the SwitchKeys

Go to PlayMap Menu > Configure > SwitchKeys/Pedal.

Set the NextEvent dropdown box to a SwitchKey or Pedal.

Setup the Sync Event

You will need to decide what type of event you want to synchronise - beats or chords.

Generally, using beats is easier, and is similar to the other sync methods.

However, you can use chord synchronisation as an alternative. This will allow you to play chords on a different beat to the default.

To set the event type, go to the Miscellaneous Config window (PlayMap Menu > Configure > Miscellaneous) and select Next Beat or Next Chord in the Live Audio Sync Event panel.

Load a Project and Play

You will need to load a Project containing a ChordFile that matches the audio that you want to play along to.

You should then set the current cursor to the bar you want to start playing at (cue point).

Start playing the audio stream on an external player. When it reaches the cue point, hit the SwitchKey (Next Event).

You can now play the first chord from the cue point.

Continue hitting the SwitchKey when the next chord or beat is reached, whilst playing the chords.

It doesn't matter if the chord is played just before or after a SwitchKey, as it will be ready to locate to the chord after the previous SwitchKey is released.

Chords will only play if the SwitchKey is pressed down. If you are playing chords or a slow song, make sure you don't release the SwitchKey too early!
