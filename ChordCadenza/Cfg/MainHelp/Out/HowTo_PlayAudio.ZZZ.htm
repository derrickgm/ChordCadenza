How To Play along with an Audio File, Audio Stream, or Live Audio

You can play along to an audio file (MP3 etc) or an audio stream in a similar way to a midifile, but with some important differences:

  • You can't create a ChordFile directly from an audio file - you will need to do this from a midifile that approximately matches the audio file, or from sheet music.
  • You can't mute tracks as you can with a midifile. However, you can download backing tracks that usually allow you to mute or solo audio tracks.
  • You will need to synchronize the midi data (chords) to the audio file using the synchronisation function, or synchronise as you play along.

    • dfvkdfk dfkfdlkdflj kjkh
    • djhds qwerty dndjkn dfsjkdjksd xzcksdck

      • Choose a Synchronisation Method

      You can synchronise to the audio stream in one of three ways:

      • File Synchronisation
      • Stream Synchronisation
      • Live Synchronisation

        • Use the How to Sync to an Audio Source window to choose between the three methods above.

        If you choose the Live Syncronisation, you should go to the HowTo Play along to Live Audio window, otherwise continue here.

        Setup the Sync SwitchKey

        Goto PlayMap Menu > Configure > KeySwitches/Pedal.

        Set the Sync dropdown box to a SwitchKey or Pedal.

        Load a Song

        If you have already created a SyncFile (.chtc or .chtx), this will be loaded as part of the Song.

        Creating a SyncFile is covered in another HowTo ( How To Sync to an Audio Source ).

        After loading a Song, the Play Audio button on all playable windows ( TrackMap, ChordMap and PlayMap ) will be activated if a SyncFile has also been automatically loaded.

        Start Sync Play

        Click The Play Audio button on any playable window ( PlayMap, TrackMap, or ChordMap ).

        If you are using an audio file, it will start playing from the current cursor position, or from the last recorded sync point before this.

        Otherwise, if you are using an audio stream, you will need to start it at some point before the current cursor position, and hit the sync key when it reaches the current cursor position.

        (If the AudioSync window is open and the option to Start from Position Time is enabled and checked, it will start from the Time on the Position slider.)

        The position will be updated on all visible playable windows as the audio file plays, and you should be able to use the ChordMap to play along with it in the same way as you play along with a midifile.
