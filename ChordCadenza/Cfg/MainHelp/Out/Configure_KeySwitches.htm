Configure KeySwitches or Pedal

KeySwitches are used to allow actions to be taken from a key press on the midi keyboard or pedal.

You can find out how to set up and use keyswitches here

The action of a particular keyswitch is explained in the documentation covering the function where the keyswitch is used. Here is a brief summary of each keyswitch:

KBChord: Play the nearest chord note to the played note whilst switchkey is down. If nearest note is already sounding, play the next chord note up or down.

Sustain: Activate sustain whilst switchkey is down. This is the normal action for the sustain pedal.

Mode: Switch between KBMode and ChordMode when switchkey is pressed down.

Sync: used for synchronizing the chords to an audio source.

Next Event: move forward to next chord.
