PlayMap Help Menu

Map Check Item

If checked, the top area of the PlayMap will show the bar numbers, lyrics (if present), chord names, and MidiFile notes (if present).

This can be useful when playing unsynchronised audio files - see How To Play Along with Live Audio.

Controls Check Item

If checked, a window containing various controls is shown.

This should normally be checked, but it may be useful to uncheck it if you want to see more of the keyboard displays at the bottom.
