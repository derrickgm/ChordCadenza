Manual Chord Synchronisation Window

This is used to configure the parameters used when playing along with an unsynchronised audio source using chord positioning. It is typically used when playing along with an internet stream or other musicians. It can also be used to play along with an audio file (.mp3), whether it is synchronised or not.

The window is displayed by selecting Configure > Manual Chord Sync from the PlayMap.
