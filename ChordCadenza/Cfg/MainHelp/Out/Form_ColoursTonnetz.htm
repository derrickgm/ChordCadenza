Tonnetz Colours

This window can be used to set the colours used by the Tonnetz Display.

Configurable Colours

The colours set by this window are:

  • Background:

    Change colour of Tonnetz background.

  • Diatonic Note:

    Change colour of font used for diatonic notes.

  • Chromatic Note:

    Change colour of font used for chromatic notes.

  • Active Node:

    Change colour of nodes representing currently active notes in the ChordMap.

  • Inactive Node:

    Change colour of nodes representing currently inactive notes in the ChordMap.

  • Played Note:

    Change colour of nodes selected by a mouse click.

  • New Active Node:

    Change colour of nodes representing notes that are about to become active (using look ahead function).

  • Old Active Node:

    Change colour of nodes representing notes that are about to become inactive (using look ahead function).
