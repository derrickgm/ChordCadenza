TrackMap CheckBoxes

Lyrics

Display Lyrics Panel, using data from the LyricsFile.

Similar to the Lyrics Panel in the ChordMap., but not editable.

Even Trk Spacing

If checked, each track will be about the same height, but the pitch scale will vary.

If unchecked, the pitch scale will be constant, but the track height will vary.

Changing this will reset the scale to its default value.

Show Beats

Check to display beat lines as well as bar lines for each track.

Show KB

Check to show midi keyboard black notes (default colour grey) for each track.

Show One Octave

If checked, will show notes condensed into one octave. This can make it easier to recognise chords.

Otherwise, actual pitches will be shown over whatever range each track uses.
