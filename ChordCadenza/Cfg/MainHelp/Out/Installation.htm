Installation

System Requirements

Windows Vista SP1, windows 7, 8, 8.1, 10

.Net 4.0 (installed by default on Windows 8 and later. May require a Microsoft Update for Windows Vista SP1 and Windows 7. The application will check this when first run.)

Audio output hardware (sound card, speakers or headphones)

Access to some MidiFiles. (If you haven't got any MidiFiles, there are many thousands available on the internet. Just do a search for MidiFiles.)

A midi keyboard is required, unless you only intend to use the key and chord generate functions of the program.

Install

Run the Setup ChordCadenza.exe file. If a previous version of Chord Cadenza is already installed, a warning message will be displayed before removing it. The option to install the starter soundfont file should be checked, unless you are going to be using an external synthesizer to convert midi to audio, or you already have a soundfont installed.
