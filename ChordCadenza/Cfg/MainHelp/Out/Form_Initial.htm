Initial Window

This is displayed immediately after the program starts for the first time, and whenever the program starts again, until the option to not show again is taken.

It has the following buttons:

Close Button

Close this window

Close Window and Don't Show Again Button

Close this window and don't show again

Configure Audio Devices Button

Configure ASIO and non_ASIO Soundcards

Configure Midi Device and SoundFonts Button

Configure Midi Input (Keyboard) and Output (Keyboard and MidiFile Player) Devices

Choose SoundFont file

Adjust Volume and Fine Tuning

Configure Special Effects (FX)

Configure Keyboard Ranges Button

Set Highest and Lowest playable notes on the Midi Keyboard

Set PlayMap display range

Introduction Button

Show Introduction Help screen.

Save Configuration Button

Save Global Configuration Parameters

Saves any changes made to various global parameters (as opposed to parameters only associated with the current Project).

This involves saving the following files in the application directory:

  • ChordCadenza.ini (the main config file)
  • Switch.ini (SwitchKeys)
  • frmNMColors.ini (ChordMap colours)
  • frmSCColors.ini (PlayMap colours)
  • ChordCfg.ini (Chord generation parameters)
  • Chords.Ranks.ini (Chord Rank values)
