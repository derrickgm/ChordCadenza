Tonnetz Window

The Tonnetz window can be used to show and play chords. It is based on the Tonnetz (Tone Network) diagram first described by Leonhard Euler in 1739.

Each node of the diagram represents a note of the chromatic scale. Each row contains notes separated by a perfect fifth interval. The notes on the diagonals are a major third and a minor third apart.

Major chords are represented as a triangle pointing up. A minor chord is a triangle pointing down. Augmented chords are 3 nodes on a diagonal (lower left to upper right). A diminished chord is repesented by 3 nodes on diagonal (upper left to lower right). Chords with 4 notes are also represented by connected nodes.

Notes that are diatonic to the current key are shown with a bold font. Chromatic notes are shown with a normal font.

As the program plays a MidiFile, the diagram is updated to show the chords present in the ChordMap. It shows chords that are approaching, chords that are currently active, and chords that are about to be ended. This allows you to see what is happening, or about to happen, in real time.

If you click on somewhere in the diagram, it will change the current chord. This will allow you to play a different chord on the Midi Keyboard to the current chord.

Clicking with the left mouse button on an inactive node will add that node to the current chord. Clicking with the right mouse button on an inactive node will replace a note in the current chord with the clicked note. The note replaced is selected by an algorithm as the most dissonant or chromatic note.

Clicking on an active node will remove that note from the curent chord.

Clicking with the left mouse button in between nodes will change the chord to a major or minor chord, consisting of the nearest 3 nodes to the clicked position. Clicking with the right nouse button in between nodes will play an augmented chord or a diminished seventh chord, depending on which diagonal is nearest.

To take full advantage of the play features of the Tonnetz display, you should set KBChord mode on, or set the Play Mode to Chord.

KBChord can be used by setting the switch key KBChord (see Setting Switch Keys.) It is best to use the sustain pedal as a switch key, which can be combined with the normal sustain function. Alternatively, you can set the KBChord During Play CheckBox to keep KBChord mode permamnently on.

Chord Play Mode can be set using the Chords Option Button in the Play Mode panel of the PlayMap.
