Configure Bezier Window

The contents of the window include:

Bezier Curve Display

Graph showing the Velocity Profile or Aftertouch Profile.

The horizontal axis is the Velocity or Aftertouch value output by the midi keyboard.

The vertical axis is the Velocity or Aftertouch value sent to the midi output.

The red circles at the end of the straight line can be dragged by the mouse to change the curve.

Load Default Values Button

Load default profile - a straight line representing no change in Velocity or Aftertouch.

Close Button

Close this window
