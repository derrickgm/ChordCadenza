Tonnetz Display Compoenets

Circle Outline Buttons

Adjusts the width of the circular outline on each node.

Spacing Buttons

Adjusts the spacing between nodes.

Diameter Buttons

Adjusts the size of the nodes.

Look Ahead Numeric Input

Set the number of beats to look ahead. Affects the display of circular rings for approaching chord notes, and chord notes that are about to finish.

Key Display

Shows the current key.

Colours Button

Opens a Dialog to set various colours on this window.

This Chord Display

Shows the current chordname and qualifier from the ChordMap.

Mouse Chord Display

Shows the chordname and qualifier selected by the mouse on the Tonnetz display.

Next Chord Display

Show the next chordname and qualifier to be played from the ChordMap.
