How To Play using the PC Keyboard

If you don't have a Midi keyboard, you can still play notes and chords using the PC Keyboard. This is not meant as a replacement for a Midi keyboard, but you can do most of the things that a Midi keyboard can do.

Configuration

The default keyboard mapping should work with standard UK and US keyboards. It defines a 2 octave range, using only alphanumeric keys. If you have a different type of keyboard, or want to map to different keys, you can select the PlayMap Menu > Configure > PCKB Keys to set up to 4 octaves. Most keys can be mapped, with the exception of the 'Print Screen' and 'Windows Logo' keys.

If the Play Notes per Chord (PC KB) CheckBox in the Miscellaneous Configurations Window is checked, this will allow you to play chords with a single key press. This is usually better than trying to play a chord with multiple key presses on a PC keyboard.

If you are using any special keys (e.g. Shift, Control, Alt, Caps Lock), then any options involving 'Sticky Keys', 'Toggle Keys', or 'Filter Keys' should be turned off in the Windows Control Panel or Settings. (On Windows 10, these are located in the Control Panel > Ease of Access Center).

Using the PC Keyboard

By default, the PC keyboard will be active if there is no Midi In device connected. You can change this by clicking PlayMap Menu > Configure > Miscellaneous, and checking the Disable PCKB Checkbox.

Pressing a key will switch a note on, letting go switches it off. You can press more than one key at a time, however there is a limit to the number of simultaneous key presses that will be recognised by the system. This depends on which combination of keys are pressed and the internal wiring of the keyboard. You can often play 4 keys simultaneously.

If the PC keyboard is active and a Project is loaded, the bottom line of the PlayMap display will show the alphanumeric character corresponding to the diatonic notes of the current musical key. If the PCKBChar CheckBox in the Note Display panel in the PlayMap. is also checked, the notes on the main display will also show the PC keyboard alphanumeric characters.

The bottom note (C) of the bottom octave should always align to the bottom note of the PlayMap display (unless the keybord is transposed). SwitchKeys are mapped to the function keys on the PC keyboard. Each of the 12 function keys are mapped to the corresponding midi SwitchKey (F1=C, F2=C# ... F12=Bb). You can define SwitchKeys to increase or decrease the keyboard velocity using PlayMap Menu > Configure > Switch Keys. You can also adjust the velocity with the mouse wheel, provided the mouse pointer is located over the main panel on the PlayMap. (You don't need to click the mouse first.)

Comparison with Midi Keyboard

The PC keyboard should be as responsive as a Midi keyboard. The main differences are:

  • The keyboard velocity has to be set to a constant value - you can't make a note louder by hitting the keyboard harder!
  • There is a limit to the number of keys you can play simultaneously.
  • There is no sustain pedal or switch on a PC keyboard. However, you can set a SwitchKkey to perform sustain, using PlayMap Menu > Configure > Switch Keys
  • There is no mod wheel or pitch wheel on a PC keyboard.
  • It's more difficult to work out where the notes are on a PC keyboard. A Midi keyboard is always going to feel more natural.

Using the Mouse

You can also play notes by clicking with the mouse on the main panel of the PlayMap. You can only play single keystrokes this way, and it is less responsive than the Midi or PC keyboards.
