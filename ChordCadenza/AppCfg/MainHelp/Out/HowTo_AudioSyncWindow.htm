How To Use the AudioSync Window

The AudioSync window is used to manage the audio sync data.

There are 2 'tracks' in this data: 'Record' and 'Play'.

The Record track is created when you sync an audio file using the 'Sync Audio' button on the ChordMap or TrackMap.

The Play track controls the current cursor when playing an audio file.

It is loaded from a .chtc file, and is updated when you copy or merge from the Record Track

Managing AudioSync Data

The AudioSync window can be used to:

  • Merge Recorded data to the Play track
  • Copy Recorded data to the Play track
  • Clear The Record or Play tracks
  • Remove data from the Play or Record track
  • Save the Play track to the .chtc file
  • Adjust the audio file sound volume and balance
  • Show and set the current audio file position
