Load Project Components

Filename Extensions ListBox

Shows a list of ChordFile filename extensions present on disk associated with the Project.

Selected name will be used as the loaded ChordFile.

Click Load File Button, or doubleclick the selected filename, to load a Project containing the selected filename.

Delete Button

Click to delete the selected file.

Load Project (or Save File) Button

Click to load a Project containing the selected filename, or to save the ChordFile to the selected filename.

Cancel Button

Close this Window without taking any action.
