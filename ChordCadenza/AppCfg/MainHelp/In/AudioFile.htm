The Audio File

This is an optional file that may be present in the Project folder. You can play along to this using file synchronisation - see How To Sync to an AudioFile.

The file can be in any of the following standard formats:

  • *.mp1 - MPEG layer 1
  • *.mp2 - MPEG layer 2
  • *.mp3 - MPEG 1.0, 2.0 and 2.5 layer 3
  • *.wav - PCM format
  • *.riff - PCM format
  • *.aiff - Audio interchange File Format
  • *.ogg - open container format
