Getting Started

Start the Application

You should run the application as a normal user.

The first screens you should see are the Initial Configuration window and PlayMap window.

You can reopen the Initial Configuration Window from the PlayMap Menu > Help > Show Initial Screen.

Initial Configuration Window

This contains links to various configuration tools that you will need to set before using this application.

Click the button to configure Midi input and ouput devices.

If you have a Midi Keyboard, click the button to configure the Midi Keyboard ranges.

If you intend to use the built-in synthesizer of this app, or to play audio files, click the button to confgure audio devices.

These configurations can be changes at any time from the PlayMap Menu.

Create a New Project

Go to the PlayMap Window and click the New Project Button.

This will allow you to copy an existing MidiFile into a new Project. If you haven't got any MidiFiles, you can download one from the internet - just do a search for MidiFiles.

You will also need to decide where to put the project, and what to call it.

Create a ChordFile

You will need to create some chords in order to use the chord play functions of this application.

You can either Generate Chords or Insert Chords

The chords will be saved in a ChordFile (*.chp) as part of the Project.

Play the Chords

You can now Play the Chords along with the MidiFile, or Play a Melody along with the MidiFile
