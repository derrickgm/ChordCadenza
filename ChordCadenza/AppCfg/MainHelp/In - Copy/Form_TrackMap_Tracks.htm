TrackMap Tracks

Each track of the TrackMap contains a Control Panel and a Display Panel.

The Control Panel

This shows basic information about the track, and also allows various properties to be changed.

It contains these controls:

Top Row

Up or Down Arrow

$$frmTrackMap.$Trk:chkcollapse

Track Description

$$frmTrackMap.$Trk:txttitle

Style Display

$$frmTrackMap.$Trk:lbltrktype

Middle Row

Channel Number Display

$$frmTrackMap.$Trk:lblchan

Instrument Name

$$frmTrackMap.$Trk:cmb

BottomRow

Check Box

$$frmTrackMap.$Trk:chk

'M' Button

$$frmTrackMap.$Trk:mute

'S' Button

$$frmTrackMap.$Trk:solo

Track Number Display

$$frmTrackMap.$Trk:lbltrk

Slider

$$frmTrackMap.$Trk:trkbar

The Trk Panel

$$frmTrackMap.$Trk:pic
